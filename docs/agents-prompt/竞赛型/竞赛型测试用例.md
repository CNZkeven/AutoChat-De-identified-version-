# 竞赛型智能体测试用例

## 用例 1：指定竞赛咨询
- 目的：验证学院历史+赛事信息查询
- 输入：
  - 角色：高校学生
  - 提问："我想参加挑战杯，学院往年成绩怎么样？"
- 侧重点：
  - 是否调用 competition_history + competition_info
  - 是否输出赛事画像与学院历史

## 用例 2：推荐竞赛
- 目的：验证推荐与匹配度分析
- 输入：
  - 角色：高校学生
  - 提问："我是自动化专业，想找适合的学科竞赛，有推荐吗？"
- 侧重点：
  - 是否给出匹配度分析与路线建议
  - 是否追问时间/基础

## 用例 3：技能准备
- 目的：验证技能推断
- 输入：
  - 角色：高校学生
  - 提问："备赛机器人类竞赛需要学什么？"
- 侧重点：
  - 是否调用 skill_inference
  - 是否给出分阶段学习建议

## 用例 4：导师与组队
- 目的：验证导师/团队模型工具调用
- 输入：
  - 角色：高校学生
  - 提问："能推荐指导老师吗？组队有什么建议？"
- 侧重点：
  - 是否调用 recommend_advisor + analyze_team_model
  - 是否给出团队结构建议

## 用例 5：时间与阶段规划
- 目的：验证阶段性备赛路线
- 输入：
  - 角色：高校学生
  - 提问："比赛还有三个月，怎么安排备赛节奏？"
- 侧重点：
  - 是否输出阶段规划与行动清单
  - 是否提出补充问题
