# 探究型智能体测试用例

## 用例 1：概念机制类问题
- 目的：验证知识库调用与启发式引导
- 输入：
  - 角色：高校学生
  - 提问："为什么闭环系统更稳定？"
- 侧重点：
  - 是否调用 search_knowledge_repository
  - 是否用问题引导而非直接结论

## 用例 2：研究起点与问题聚焦
- 目的：验证问题澄清与路径建议
- 输入：
  - 角色：高校学生
  - 提问："我想研究无人船避碰路径规划，从哪开始？"
- 侧重点：
  - 是否提出澄清维度（场景/目标/评价标准）
  - 是否给出探索路径

## 用例 3：概念对比
- 目的：验证比较类问题的引导
- 输入：
  - 角色：高校学生
  - 提问："状态空间方法和传递函数方法有什么本质区别？"
- 侧重点：
  - 是否调用 search_knowledge_repository
  - 是否通过对比维度引导

## 用例 4：方向偏离与反思
- 目的：验证重新聚焦与反思引导
- 输入：
  - 角色：高校学生
  - 提问："我感觉方向偏了，怎么回到核心问题？"
- 侧重点：
  - 是否引导学生回到核心问题
  - 是否给出可执行的回归步骤

## 用例 5：实验验证指标
- 目的：验证实验设计引导
- 输入：
  - 角色：高校学生
  - 提问："准备做实验验证，需要哪些评价指标？"
- 侧重点：
  - 是否给出指标维度而非直接答案
  - 是否提出补充问题
