# 课程型智能体测试用例

## 用例 1：课程难度与内容
- 目的：验证课程信息查询与学习路径建议
- 输入：
  - 角色：高校学生
  - 提问："自动控制原理难吗？主要学什么？"
- 侧重点：
  - 是否调用 curriculum + tech_evolution
  - 是否给出学习路径与重点难点

## 用例 2：选修课推荐
- 目的：验证选修推荐与匹配度
- 输入：
  - 角色：高校学生
  - 提问："我是测控大二，想选编程相关的选修课，有推荐吗？"
- 侧重点：
  - 是否调用 curriculum
  - 是否追问学分/时间约束

## 用例 3：学分缺口与毕业要求
- 目的：验证学分差距提示

  - 是否说明培养要求与缺口
